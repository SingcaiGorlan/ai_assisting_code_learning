# Wails 桌面应用功能概览

## 📱 应用界面

AI 辅助代码学习平台桌面版提供了现代化、直观的用户界面。

### 1. 登录页面
- **渐变背景**: 蓝紫色渐变,视觉效果出色
- **卡片设计**: 毛玻璃效果,现代感十足
- **表单验证**: 实时输入验证
- **双模式**: 支持登录和注册切换

### 2. 仪表板 (Dashboard)
- **统计卡片**: 
  - 📚 学习课程数量
  - ⏱️ 累计学习时长
  - 📈 整体完成进度
  - 💬 AI 对话次数
  
- **最近学习**: 显示正在学习的课程及进度
- **快捷操作**: 快速继续学习和访问 AI 助手

### 3. 课程页面
- **网格布局**: 响应式卡片布局
- **进度追踪**: 可视化进度条
- **课程信息**:
  - 课程标题
  - 详细描述
  - 完成章节/总章节
  - 学习百分比

### 4. AI 助手
包含两个功能模块:

#### AI 对话
- 实时聊天界面
- 消息历史记录
- 智能回复
- 加载状态提示

#### 代码辅助
- 代码粘贴区域
- 问题输入框
- AI 建议展示
- 代码优化指导

## 🎨 设计特色

### 配色方案
- **主色调**: 紫色系 (#8B5CF6 - #7C3AED)
- **辅助色**: 蓝色系 (#3B82F6 - #2563EB)
- **渐变色**: 蓝紫渐变、紫粉渐变
- **中性色**: 灰色系用于文本和边框

### UI 组件
- **按钮**: 渐变背景 + 圆角 + 阴影 + 悬停动画
- **卡片**: 白色背景 + 圆角 + 阴影 + 悬停效果
- **输入框**: 边框 + 聚焦高亮 + 图标装饰
- **导航栏**: 固定顶部 + 图标菜单

### 动画效果
- 按钮悬停缩放
- 卡片阴影过渡
- 加载旋转动画
- 页面切换过渡

## 🔧 技术实现

### 前端技术
- **React 18**: 组件化开发
- **TypeScript**: 类型安全
- **Tailwind CSS**: 原子化 CSS
- **Lucide Icons**: 精美图标库
- **React Router**: 客户端路由

### 后端技术
- **Wails v2**: 桌面应用框架
- **Go 1.21+**: 系统级编程
- **Runtime API**: 系统功能调用

### 架构优势
```
┌─────────────────┐
│   React 前端    │
│  (UI 界面层)    │
└────────┬────────┘
         │ Bridge
┌────────▼────────┐
│   Go 后端       │
│ (业务逻辑层)    │
└─────────────────┘
```

## 📊 性能特点

| 特性 | Wails | Electron |
|------|-------|----------|
| 应用体积 | ~10MB | ~150MB |
| 内存占用 | ~50MB | ~200MB |
| 启动速度 | 快 | 较慢 |
| 渲染引擎 | 系统 WebView | Chromium |
| 跨平台 | ✅ | ✅ |

## 🚀 快速体验

```bash
cd web/wails-app
make install  # 首次运行
make dev      # 启动开发模式
```

应用将自动打开,体验原生桌面应用的流畅感!

## 📦 构建发布

```bash
# 构建当前平台
wails build

# 跨平台构建
wails build -platform windows/amd64
wails build -platform darwin/amd64
wails build -platform linux/amd64
```

输出目录: `build/bin/`
