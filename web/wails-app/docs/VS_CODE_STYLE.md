# VS Code 风格界面设计

## 🎨 设计概述

本项目采用了类似 VS Code 的界面风格，提供专业、现代的开发者体验。

## 📐 界面布局

```
┌─────────────────────────────────────────────────────┐
│  活动栏  │  侧边栏        │  标签栏                  │
│  (48px)  │  (250px)       │  TabBar                 │
│          │                ├─────────────────────────┤
│  📄      │  导航菜单      │                         │
│  🔍      │  - 首页        │   内容区域               │
│  🔀      │  - 课程        │   (Pages)                │
│  🐛      │  - AI 助手     │                         │
│  📦      │  - 设置        │                         │
│          │                │                         │
│  👤      │  ┌──────────┐  │                         │
│        │  │ 用户信息  │  │                         │
└──────────┴──┴──────────┴──┴─────────────────────────┘
│              状态栏 (StatusBar)                      │
─────────────────────────────────────────────────────┘
```

## 🎨 配色方案

### 核心颜色
- **主背景色**: `#1e1e1e` - 深灰色背景
- **侧边栏背景**: `#252526` - 稍浅的灰色
- **活动栏背景**: `#333333` - 更浅的灰色
- **标签栏背景**: `#2d2d2d` - 中灰色
- **边框颜色**: `#3c3c3c` / `#252526` - 边框分隔线

### 强调色
- **选中状态**: `#094771` - 深蓝色背景
- **悬停状态**: `#2a2d2e` - 浅灰悬停
- **活动指示器**: `#007acc` - VS Code 蓝色
- **状态栏**: `#007acc` - 蓝色状态栏

### 文字颜色
- **主文字**: `#ffffff` - 白色
- **次要文字**: `#gray-400` / `#9ca3af` - 灰色
- **链接/按钮**: `#007acc` - 蓝色

## 📦 组件说明

### 1. Sidebar（侧边栏组件）

位置：`frontend/src/components/Sidebar.tsx`

**功能特性：**
- 左侧活动栏（48px 宽度）
  - 资源管理器图标
  - 搜索图标
  - 源代码管理图标
  - 运行和调试图标
  - 扩展图标
  - 底部：账户和退出按钮

- 右侧导航菜单（250px 宽度）
  - 导航标题
  - 菜单项（首页、课程、AI 助手、设置）
  - 用户信息卡片

**样式特点：**
- 选中项蓝色背景高亮
- 悬停效果平滑过渡
- 活动图标左侧白色指示条

### 2. TabBar（标签栏组件）

位置：`frontend/src/components/TabBar.tsx`

**功能特性：**
- 页面标签显示
- 当前活动标签高亮
- 点击切换页面
- 关闭按钮（视觉）

**样式特点：**
- 35px 高度
- 深灰色背景
- 活动标签浅色背景

### 3. StatusBar（状态栏组件）

位置：`frontend/src/components/StatusBar.tsx`

**功能特性：**
- 左侧：Git 分支、错误数、警告数
- 右侧：行号、缩进、编码、文件格式

**样式特点：**
- 22px 高度
- 蓝色背景（#007acc）
- 白色文字
- 悬停高亮效果

### 4. Dashboard（仪表板页面）

位置：`frontend/src/pages/Dashboard.tsx`

**功能特性：**
- 面包屑导航
- 欢迎区域
- 统计卡片网格（4 个）
- 最近学习列表
- 快捷操作按钮

**样式特点：**
- VS Code 深色主题
- 卡片边框悬停效果
- 进度条渐变色彩
- 响应式网格布局

##  设计亮点

### 1. 专业感
- 采用 VS Code 经典配色
- 统一的深色主题
- 精确的间距和尺寸

### 2. 交互体验
- 平滑的过渡动画
- 清晰的悬停反馈
- 直观的选中状态

### 3. 响应式设计
- 支持不同屏幕尺寸
- 网格布局自适应
- 滚动条自定义样式

### 4. 可访问性
- 清晰的文字对比度
- 合理的字体大小
- 图标与文字结合

## 📝 使用指南

### 启动开发模式

```bash
cd web/wails-app
make dev
```

或

```bash
wails dev
```

### 构建生产版本

```bash
wails build
```

## 🔧 自定义指南

### 修改配色

编辑 `styles.css` 和各个组件中的颜色值：

```css
/* 主背景色 */
bg-[#1e1e1e]

/* 侧边栏背景 */
bg-[#252526]

/* 活动栏背景 */
bg-[#333333]

/* 状态栏蓝色 */
bg-[#007acc]
```

### 添加新导航项

编辑 `Sidebar.tsx`：

```typescript
const navItems: SidebarItem[] = [
  // ... 现有项
  { 
    icon: <NewIcon size={18} />, 
    label: '新功能', 
    path: '/new-feature',
    tooltip: 'New Feature'
  },
]
```

### 修改统计卡片

编辑 `Dashboard.tsx`：

```typescript
const stats = [
  { icon: <Icon />, label: '标签', value: '数值', color: '#颜色' },
  // ...
]
```
