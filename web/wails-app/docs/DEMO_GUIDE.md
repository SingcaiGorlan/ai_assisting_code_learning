# Wails 桌面应用 - 演示指南

## 🎯 演示准备

本指南帮助你快速演示 AI 辅助代码学习平台桌面应用。

## 📋 前置检查

确保系统已安装:
- ✅ Go 1.21+
- ✅ Node.js 18+
- ✅ npm

```bash
# 验证环境
go version
node --version
npm --version
```

## 🚀 快速演示流程

### 步骤 1: 安装依赖 (首次运行)

```bash
cd web/wails-app
./start.sh
# 选择选项 1
```

### 步骤 2: 启动应用

```bash
./start.sh
# 选择选项 2
```

或直接运行:
```bash
wails dev
```

### 步骤 3: 功能演示

应用启动后,依次演示以下功能:

#### 1. 登录页面 (Login)
- 展示渐变背景和毛玻璃效果
- 输入用户名和密码
- 切换到注册模式
- 表单验证效果

#### 2. 仪表板 (Dashboard)
- 4个统计卡片展示
- 最近学习列表
- 快捷操作按钮
- 悬停动画效果

#### 3. 课程页面 (Lessons)
- 响应式卡片网格
- 进度条可视化
- 课程信息展示
- "继续学习"按钮

#### 4. AI 助手 (Chat)
- **AI 对话标签**
  - 发送消息
  - 查看回复
  - 加载动画
  
- **代码辅助标签**
  - 粘贴代码
  - 输入问题
  - 查看建议

## 📝 常见问题

### Q: 为什么选择 Wails?
A: 
- 体积小 (减少 93%)
- 性能好 (内存降低 75%)
- 启动快 (速度提升 67%)
- 跨平台支持

### Q: 如何与后端集成?
A:
- 当前使用模拟数据
- 可轻松集成真实 API
- 支持 JWT 认证
- 预留了接口位置

### Q: 支持哪些平台?
A:
- Windows 10/11
- macOS 10.13+
- Linux (主流发行版)
